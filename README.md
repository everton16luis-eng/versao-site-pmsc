# versao-site-pmsc
🌐 Projeto – Programação para Internet I

Projeto desenvolvido como parte da avaliação da disciplina de Programação para Internet I, no curso de Análise e Desenvolvimento de Sistemas do IFSC – Campus Canoinhas (SC).

📌 Sobre o Projeto

Este projeto tem como objetivo aplicar na prática os conceitos fundamentais de desenvolvimento web abordados durante a disciplina, incluindo:

Estruturação de páginas com HTML
Estilização com CSS
Interatividade com JavaScript
Organização de layout responsivo
Boas práticas de desenvolvimento front-end

O sistema simula um ambiente real de aplicação web, permitindo consolidar conhecimentos essenciais para o desenvolvimento de interfaces modernas.

🚀 Funcionalidades
📄 Estrutura de páginas web organizada
🎨 Layout estilizado e responsivo
📱 Adaptação para diferentes tamanhos de tela
⚡ Interações dinâmicas com JavaScript
🔍 Navegação intuitiva
🛠️ Tecnologias Utilizadas
HTML5
CSS3
JavaScript
